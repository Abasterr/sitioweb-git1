## Comandos GIT

### 1. Para inicializar un proyecto de git 

```sh
git init
```

### 2. El comando para ver el estado de tu proyecto

```sh
git status
```

### 3. Para dar siguimiento y agregar a la zona del stage
 
En caso de agregar todo los archivos sin siguimiento y archivos modificados a la zona del stage

```sh
git add .
```

En el caso que desees agregar un archivo especifico

```sh
git add index.html
```
